# TEL_Pico

使用 Pico 控制 TEL 機器人

## 文件說明

-   Brushless_motor : 無刷馬達控制程式 (尚未決定 PICO 與通訊)
-   CLI_move : 使用 CLI 控制移動平台
-   DBR4_Chanel_test : DBR4 收訊頻道測試
-   Lifting_platform : Arduino 升降平台程式
-   Reciver : 收訊程式與發送訊號
-   Remote_move : 移動平台控制程式
-   Servo_motor : 伺服馬達控制程式(俯仰、水平)
